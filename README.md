# QA Automation Framework

This project demonstrates a complete QA automation workflow including UI automation testing and API testing.

## Tools Used
- Python
- Selenium WebDriver
- PyTest
- Postman
- Git & GitHub

## Project Structure
tests/ui_tests – Selenium automation tests  
tests/api_tests – API tests  
pages – Page Object Model implementation  
postman_collections – Postman API collections  
reports – Test execution reports  

## Test Scenarios
- Website loading
- Login functionality
- Add to cart
- Checkout workflow
- Product API testing
